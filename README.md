# 💪 Duo Energy — Fitness Tracker

Aplicación de seguimiento de entrenamiento para Sofía y Ángel con sincronización en tiempo real.

## ⚡ Características

- ✅ Rutinas personalizadas por atleta
- ✅ Tracking de peso corporal con gráficos
- ✅ Sincronización en tiempo real con Firebase
- ✅ Notas por ejercicio
- ✅ Timer automático de descanso
- ✅ Backup y exportación de datos
- ✅ Modo claro/oscuro
- ✅ Funciona offline

## 🚀 Cómo usar

### Primera vez
1. Abre la app
2. Si Firebase no está configurado, aparecerá un banner
3. Pega tu API Key, Database URL y Project ID (del proyecto Firebase)
4. ¡Listo! Los datos se sincronizan automáticamente

### Desde el celular
- Sofía y Ángel usan la misma URL
- Cada uno selecciona su nombre
- Los datos se sincronizarán en tiempo real si ambos tienen Firebase configurado

## 📊 Datos almacenados

- Ejercicios completados
- Pesos y historial
- Notas por sesión
- Peso corporal semanal
- Configuración personal

## 🔐 Privacidad

- Los datos se guardan en Firebase (tu proyecto)
- Backup local en localStorage
- Exporta tus datos cuando quieras
